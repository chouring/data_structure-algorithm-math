# 动态规划dp
## 概念
    动态规划的思想是：通过解决相同类型更小规模问题，来解决更大规模的问题。也就是数学中“递推”思想的进一步应用。
## 功能
    实际上可以理解为对答案空间的“分类”枚举，（搜索算法是暴力枚举，那么dp就是分类枚举。因此dp效率往往比搜索高）
    dp根据不同类型的问题分为许多类型，至少有如下几种类型。
