# 双指针
## 概念
    两个指针线性移动
    
## 功能
    可以视为对暴力枚举的优化，时间复杂度O(N)

## 模板
```cpp
// 分两种逻辑
// 1.如果i和j逻辑地位相等
int i = 0, j = 99;
while (i < j) {
    //if (/*具体逻辑*/) 更新i;
    //else 更新j;
}


// 2.如果分主被动
// i为主动指针,j为伴随指针，条件触发才维护j
for(int i = 0, j = 0; i <= r; i++) {
    //if(/*某个条件*/) 更新j;
    //常规操作
}
```
